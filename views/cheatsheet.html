{{define "cheatsheet.html"}} Special Environment Variables:

- PAYLOAD_DATA = Request body data for the webhook, whole payload will be assigned to text.

- FORM_{NAME} = Parsed form data. Each field is accessible with FORM_NAME. i.e FROM_COMMENT if your submission has comment field

- URL_PARAM_{NAME} = Parsed from url params for GET requests only. Each field is accessible with URL_PARAM_NAME. i.e URL_PARAM_ID if your url has `?id=1`

- HEADER_{HEADER_NAME} = Header name e.g. HEADER_USER_AGENT

- CRED_{CREDENTIAL_NAME} = Credential name e.g. CRED_GITHUB_OAUTH_TOKEN

For periodic tasks, environment variables regarding request will be empty

{{end}}
